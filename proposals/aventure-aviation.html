Project Overview

This proposal outlines the development of "Aventure Aviation Automation," a web-based solution designed to streamline the procurement process for aviation parts. The system will automate the labor-intensive task of searching distributor portals (e.g., Boeing, Aviol) by ingesting Excel spreadsheets, performing autonomous searches via headless browsers, and intelligently populating the best available options back into the source files on Google Drive.

1,500,000 Total Project Value (PKR)
3 Core Modules
5 Distributor Integrations

Scope & Features

The project is divided into two major components: the **Core Application Platform** (the foundation, UI, and security) and the **Distributor Automation Modules** (the web scrapers). Select a module below to explore its specific features and user stories.

Module A: Core Application Platform

Base Infrastructure, UI, Security, and Data Management.

Key Features:

  • Authentication & RBAC (Employee, Manager, Superuser)
  • Excel/CSV Ingestion Engine
  • Google Drive Integration (Upload, Retrieve, Update)
  • Visual File Gallery (Sorted by date/type)
  • Telemetry & Activity Logging
  • User Management (CRUD operations for Managers)
  • Superuser Configuration (Portal Credentials)

User Stories:

  • As a Manager, I can create, update, or revoke employee accounts.
  • As a Superuser, I can view an audit log of all user activities.
  • As an Employee, I can upload a standard Excel template.
  • As the System, I will validate and upload the file to Google Drive.
  • As an Employee, I can view a gallery of my uploaded files.
  • As a Superuser, I can input credentials for distributor portals.

Module B: Distributor Portal Automation

Autonomous Searching, "Best Available" Logic, and Data Entry.

Key Features:

  • Navigate to 5 unique distributor portals
  • Handle secure login sessions
  • Search parts by Part Number/Description
  • "Best Available" Logic (Price, Stock, Pack Qty)
  • Write-back results to specific Excel columns
  • Decoupled services for portal independence

User Stories:

  • As the System, I will spawn a Headless Chrome instance.
  • As the System, I will search for a part number on a target portal.
  • As the System, I will compare results based on Price, Stock, and Packaging.
  • As the System, I will write the selected data back to the correct columns in the Excel file on Google Drive.
